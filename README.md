# QueChalPang (퀘찰팡)

## 프로젝트 설명
발달장애가 있는 사용자가 복지 서비스, 상담, 설문 등을 쉽고 직관적으로 이용할 수 있도록 지원하는 웹 기반 플랫폼입니다.
관리자/담당자는 예약·설문·사용자 데이터를 효율적으로 관리할 수 있으며, 사용자 친화적인 UI/UX를 제공합니다.

## 프로젝트 설치 및 실행 방법

### 사전 요구사항
# 아래 환경이 설치되어 있어야 합니다.
- Node.js 
- npm 

### 설치
```bash
# 저장소 클론
git clone https://github.com/mocha2855/yedam_QueChalPang.git

# 프로젝트 디렉토리로 이동
cd yedam_QueChalPang

# 의존성 설치
npm install
```

### 실행
```bash
# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 프로젝트 사용 방법

### 주요 기능

#### 1. 상담 예약 시스템
- 사용자는 원하는 날짜와 시간을 선택하여 상담 예약 가능
- 예약 가능한 시간대를 실시간으로 확인
- 예약 내역 조회 및 취소 기능 제공

#### 2. 설문조사 작성 및 관리
- 관리자는 다단계 구조의 설문조사 생성 가능
  - 카테고리 → 서브카테고리 → 질문 형태의 3단계 구조
- 다양한 답변 유형 지원 (객관식, 주관식, 체크박스 등)
- 설문조사 버전 관리 및 수정 이력 추적
- 조건부 질문 설정으로 맞춤형 설문 구성

#### 3. 예약 관리
- 관리자 페이지에서 전체 예약 현황 한눈에 확인
- 예약 승인/거절 처리
- 예약자 정보 및 상담 내역 관리
- 일정별 예약 현황 캘린더 뷰 제공

### 4. 지원계획서/결과서
- 담당자는 사용자에 대한 지원계획서/결과서 작성 가능
- 계획서를 관리자에 승인을 받음으로써 일처리 수월

#### 4. 관리자 페이지
- 사용자 관리 및 권한 설정
- 설문조사 통계 및 분석
- 예약 통계 대시보드
- 시스템 설정 및 공지사항 관리

### 사용자 유형별 기능
- **일반 사용자**: 상담 예약, 설문조사 참여
- **담당자**: 예약 관리, 문의관리,지원계획서/결과서
- **관리자**: 문의관리, 회원관리, 지원서/지원계획서/결과서 승인관리
- **시스템관리자**: 지원서 관리(등록/수정/삭제), 기관관리, 회원관리
